# Elysium Layout for Markdown Interoperability

Writer's first Markdown import and export convention follows the Elysium layout: story prose under `story/`, character material under `characters/`, worldbuilding entries under `worldbuilding/`, exploratory notes under `braindump/`, and project-level guidance files mapped into writing briefs or notes. This supports the proven local-agent workflow immediately while leaving configurable import/export profiles for later.
