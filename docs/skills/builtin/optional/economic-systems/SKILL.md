# $economic-systems

Economic worldbuilding for authors who want resource logic, labor structures, markets, and inequality to create believable pressure instead of vague backdrop.

## Use When

- A story depends on scarcity, debt, trade, class, or extraction.
- The world has money, exchange, labor, or post-scarcity claims that need scrutiny.
- Characters' choices should be shaped by what the system rewards or punishes.

## Do Not Use When

- The request is mostly about religion, government, or prose-level revision.
- The author only needs decorative trade terms.
- The author wants final canon answers without exploratory modeling.

## Writer Workflow

1. Read the relevant Story Bible material and any story scenes that rely on economic pressure.
2. Identify the resource base, labor model, exchange system, and who controls access.
3. Map winners, losers, shadow systems, and the tensions those create for story.
4. Check whether the current world logic supports the desired stakes.
5. Keep exploratory models separate from confirmed canon until the author approves them.

## Writer Output Handling

- Return the economic model in chat by default.
- Create an Attached Note when the analysis serves one Chapter, one faction, or one local conflict.
- Create or update a Project Note for broader exploration.
- Propose Story Bible updates only after the author confirms currencies, classes, trade rules, or institutional facts.
- Do not use Structured Writes in this skill.

## Script Compatibility

This source skill has no required helper scripts. The Milestone 3.5 rewrite works through Writer-native system design and reviewable canon proposals.
