# world-fates

- Source path: `docs/skills/suggested/fiction/worldbuilding/world-fates/`
- Final disposition: `deferred`
- Why it is not included as a 3.5 built-in skill: the skill is built around campaign-scale fate pressure, weighted rolls, and world-state updates. Its core value depends on simulation-style runtime behavior and shared-world tracking that Writer does not provide in Milestone 3.5.
- What would need to change for reconsideration: Writer would need audited script runtime support, database-backed world-state inputs, and a clearer product commitment to campaign or shared-universe simulation workflows.
- Scripts/assets involved: yes. The source folder includes multiple scripts under `scripts/`, JSON data under `data/`, and templates under `templates/`.
- Future roadmap mapping: partial. Milestone 3.6 could supply the script runtime foundation, but broader campaign/shared-world support would still need its own roadmap decision.
